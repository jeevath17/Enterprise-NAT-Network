# 🌐 Enterprise NAT Network Simulation Using Cisco Packet Tracer

## 📖 Project Overview

This project showcases the design and implementation of an enterprise NAT network using Cisco Packet Tracer. It simulates communication between multiple branch offices and public servers through an ISP backbone by implementing different Network Address Translation (NAT) techniques.

The network incorporates Static NAT, Dynamic NAT, PAT (NAT Overload), RIP Version 2, DNS, and HTTP/HTTPS services to replicate a real-world enterprise networking environment.

---

## 🚀 Key Features

* Enterprise branch network connectivity
* Public and Private IP addressing
* DNS Server for domain name resolution
* Static NAT implementation
* Website access using domain names
* RIP Version 2 routing protocol
* HTTP and HTTPS Web Servers
* Connectivity testing using ICMP (Ping)
* Dynamic NAT configuration
* End-to-End network communication
* ISP backbone network simulation
* PAT (NAT Overload) for multiple device internet access
